# 📊 Sales Insights Dashboard (Power BI Project)

## 📌 Project Overview

This project presents an interactive **Sales Insights Dashboard** built using Power BI to analyze business performance across different markets, customers, and products.

The dashboard helps in identifying key trends, revenue patterns, and actionable insights for better decision-making.

---

## 🎯 Objective

* Analyze sales performance across different regions and time periods
* Identify top-performing customers and products
* Understand revenue trends and seasonal patterns
* Provide business insights for improving sales strategy

---

## 📁 Project Structure

```
Sales-Insights/
```

---

## 🛠 Tools & Technologies Used

* Power BI
* Excel (Data Source)
* Data Cleaning & Transformation (Power Query)
* Data Modeling

---

## 📊 Dashboard Features

* Interactive filters (Year, Market, Customer)
* KPI metrics (Revenue, Profit, Sales Quantity)
* Region-wise and product-wise analysis
* Time-series trend analysis
* Top customers and products visualization

## 📈 Key Insights

* Top markets contribute the majority of revenue
* A small group of customers generates a large portion of total sales
* Certain months show consistent dips in sales (seasonality)
* Specific product categories outperform others

---

## 💡 Business Recommendations

* Focus on high-performing markets for expansion
* Improve strategies in low-performing regions
* Retain top customers through loyalty programs
* Optimize inventory based on high-demand products

## 📥 How to Use

1. Download the `.pbix` file
2. Open using Power BI Desktop
3. Refresh data if needed
4. Explore dashboard using filters
